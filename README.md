# AI Research Assistant

An AI-powered research assistant built with Python and the OpenRouter API.

This project is being developed step by step, with each version introducing a new AI engineering concept. The goal is not only to build a portfolio-ready application but also to understand how modern AI applications are designed and developed.

## Current Features

- ✅ AI Chat
- ✅ Conversation Memory
- ✅ System Prompt

## Tech Stack

- Python
- OpenRouter API
- OpenAI Python SDK
- python-dotenv

## Project Status

🚧 Currently under active development.
